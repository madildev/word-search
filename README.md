## ✨ What is Word Smith?

Word Smith is your personal word wizard! 🧙‍♂️ Ever found yourself stuck trying to find the perfect word for your poem, song, or creative writing project? We've got you covered!

Simply enter a word and a topic, and Word Smith will conjure up a list of related words that connect both concepts. It's like having a thesaurus that understands context! 📚

## 🎯 How It Works

1. **Enter a Word** - Type in any word that sparks your curiosity (e.g., "love", "dream", "ocean")
2. **Choose a Topic** (Optional) - Add a theme to narrow down the results (e.g., "nature", "emotions", "travel")
3. **Click "Find Related Words"** - Watch the magic happen! ✨
4. **Explore the Results** - Discover up to 10 related words that fit your criteria

## 🚀 Features

- **Smart Word Discovery**: Uses the powerful Datamuse API to find semantically related words
- **Topic Filtering**: Narrow down results by specific themes for more precise matches
- **Beautiful Modern UI**: Clean, responsive design with smooth animations
- **Quick Examples**: Pre-built example combinations to get you started instantly
- **No Account Required**: Jump right in and start exploring words!

## 🎨 Tech Stack

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with gradients, animations, and responsive design
- **JavaScript (ES6+)** - AJAX requests using XMLHttpRequest
- **Datamuse API** - Free word-finding API that powers the magic behind the scenes
- **jQuery** - DOM manipulation (legacy support)
